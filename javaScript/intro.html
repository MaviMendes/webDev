Uma introdução ao JavaScript

O que é JavaScript

Um script é um código fonte que não precisa de pré-processamento com, por exemplo, uma etapa de compilação, antes de ser executado. No contexto da Web, o script geralmente se refere a um código escrito em JavaScript. Este código é executado pelo navegador quando uma página é baixada ou é executado em resposta a um evento acionado pelo usuário.

JavaScript é uma linguagem de programação para elaborar página web dinâmicas. Mais especificamente, JavaScript permite a criação de de elementos complexos em páginas web, como por exemplo, atualizar dados dinamicamente, mostrar dados de modo interativo, fazer animações plotar gráficos animados, entre outras tarefas.

ECMAscript é a linguagem script que é a base de JavaScript. Essa linguagem script genérica foi desenvolvida pela Ecma International e hoje é padronizada, formando um padrão JavaScript para o consórcio W3C.
