Introdução a linguagem HTML

HTML é uma linguagem de marcação de documentos. HTML significa HyperText Markup Language, ou seja, uma linguagem de marcação para hipertextos. Uma linguagem de marcação, como o próprio nome diz, marca partes do texto para serem trabalhadas. Em HTML as partes de texto marcadas indicam como fazer a formatação de um texto.

Notem que HTML não é uma linguagem de programação.

Todo documento HTML é marcado usando-se etiquetas, também chamadas de tags, do seguinte modo: 

<etiqueta>...elemento_a_ser_etiquetado...</etiqueta>

Existem, como veremos, etiquetas mais complexas, especialmente aquelas para ligações hipertextos, que estão na forma: 

<etiqueta atributo=valor>...elemento_a_ser_etiquetado...</etiqueta>
e que veremos com mais detalhes mais tarde.

Em suma, HTML como linguagem de marcação, é a linguagem padrão atualmente para descrição de páginas na Web, ou seja, é a ferramenta básica para edição de conteúdos para Internet.

Normalmente, quando solicitamos um endereço Internet (URL - Uniform Resource Locator) este aponta para um computador servidor, mais especificamente para um diretório nele. Por padronização, a página chamada, ou seja, a página que é apresentada para o usuário é a página index.html.

Este documento que você está lendo foi todo escrito em HTML. Ele foi escrito num editor de programas, usando apenas caracteres do teclado e nenhuma ferramenta de edição, para que ele pudesse ser usado também como meio de aprendizado desta linguagem de marcação.

Linguagens marcadoras simples, com esta, são uma boa opção para edição de relatórios ou documentos curtos que podem ser visualizados com ajuda de qualquer navegador. Estes documentos podem ser escritos com editores simples de texto, como o Bloco de Notas do Windows, ou o Gedit para os sistemas GNU/Linux, ou similares, e independem da versão do software de edição. Assim, você estará editando texto puro, texto que não ficará obsoleto (pelo menos enquanto esta linguagem persisitir) no seu computador.

Documento básico

Exitem, no entanto, algumas etiquetas essenciais que definem a estrutura básica de um documento HTML, que são: 
<!DOCTYPE html>
<html>
<head><title>Titulo do Documento</title></head>
<body>
texto,
imagem,
links,
...
</body>
</html>

Examinando o documento acima vemos que <html> indica o início de um documento nesta linguagem e que certamente a etiqueta </html> indica o final do texto editado

A seção indicada por <head> </head> marca o que será visto no cabeçalho de uma página, ou seja, é o título da janela do visualizador, do navegador, de textos.

Para a inserção de textos no corpo do documento a etiqueta fundamental é a marcadora de parágrafos, isso mesmo, até mesmo os parágrafos devem ser marcados para serem exibidos corretamente. Logo todo parágrafo deve estar inserido entre as etiquetas <p> </p>

Estamos então prontos para editarmos o nosso primeiro documento HTML. Faça o seu. Edite um arquivo HTML com uma frase contendo o seu nome e uma breve apresentação sua. Chame este arquivo de index.html. Para visualizá-lo, carregue-o a partir de um navegador web.

As listas

Tão importante quanto os parágrafos, as listas são outro tipo de formatação muito utilizada nos textos.
Em HTML temos basicamente três tipos de listas:

  1. as listas de definição;
  2. as listas de tópicos, não numerados; e
  3. as listas numeradas.

As listas de definição

Estas listas têm o seguinte formato:

<dl>
<dt>termo a ser definido (definition term)</dt>
<dd>definição</dd>
<dt>termo a ser definido</dt>
<dd>definição</dd>
</dl>

Que produz:

termo a ser definido
definição
termo a ser definido
definição

As listas não numeradas - Listas de tópicos

Vejam o exemplo abaixo:

<ul>
<li>item da lista</li>
<li>outro item da lista</li>
<li>mais um item</li>
</ul>
O código <ul> é mnemônico para unumbered list. Assim, conhecendo o mnemônico fica mais fácil memorizar.

Estas etiquetas de lista produzem o resultado abaixo:

  • item da lista
  • outro item da lista
  • mais um item

As listas numeradas

Vejam novamente um exemplo abaixo:

<ol>
<li>primeiro item da lista</li>
<li>segundo item da lista</li>
<li>terceiro item</li>
</ol>
O código <ol> é mnemônico para ordered list.

Produzem o resultado abaixo:

  1. primeiro item da lista
  2. segundo item da lista
  3. terceiro item

Ligações - links

As ligações, ou links, são a essencia dos documentos hipertexto. São os links que promovem o direcionamento de um texto para outro documento, outra localização, ou mais tecnicamente, outro URL.

Esta etiqueta de links tem o seguinte formato: <a href = "URL_destino">âncora</a>

onde:

URL_destino
é a URL do documento de destino;
âncora
é o texto que servirá de ligação para a URL informada.

Vejamos um exemplo:
<a href="http://www.usp.br">Link para USP</a>
É equivalente à: Link para USP.

Agora retome ao seu documento de edição, ou melhor, à página que você estava editando e insira links para locais citados, por exemplo, para a cidade que nasceu e para a Unidade onde está fazendo a sua graduação.

Inserção de imagens

Como em todo texto multimídia as imagens não devem faltar num documento HTML. Para inserção de imagens devemos proceder de modo análogo as ligações absolutas, com a diferença do elemento img.
Vejamos como inserir uma imagem que está neste mesmo diretório:

<img src="URL_imagem">

No parágrafo acima, src significa source, ou seja, o arquivo fonte, a imagem e sua localização, que deverá ser fornecido para o navegador encontar a imagem. Já URL significa Uniform Resource Locator que é o endereço Internet do objeto (imagem) que se deseja mostrar, caso esta imagem não estaja no seu disco local.

Vejamos um exemplo:

A figura acima surgiu da digitação do seguinte comando:
<img src= "brasil5.png">
já que brasil5.png é um arquivo que se encontra nesta URL.

Podemos inserir nas páginas HTML arquivos nos formatos *.jpg, *.gif e *.png. Isto, óbviamente, até este momento. No futuro esperamos que os navegadores aceitem outros formatos de arquivos de imagens.

Atributos do comando de inserção de imagens

Existem uma série de atributos que podem ser adicionados após o nome da figura, ou imagem, inserida. Entre estes atributos destacamos:

alt
descreve um texto que será exibido quando a imagem não for carragada pelo browser, em casos de navegadores tipo texto ou com o carragemento de imagens desligado;
width e height
especificam a largura e a altura da imagem;
border
especifica a existência e a espessura da borda da figura;
align
especifica o alinhamento da imagem dentro do parágrafo.

No modelo genérico específicamos o atributo da seguinte forma:

<img src="URL_imagem" atributo="valor_atributo" >

Vejamos para o caso de alt

<img src="URL_imagem" alt="texto_explicativo" >

Vejamos um exemplo:

A figura acima surgiu da digitação do seguinte comando:
<img src="brasil5.png" alt="Logotipo da USP.">

Formatação lógica de caracteres

A formatação lógica de caracteres permite especificar etiquetas que discriminen os caracteres quanto a sua especificidade e não somente quanto a sua aparência. Portanto, a apresentação caracter marcado poderá depender da implementação ou do ambiente já que não é descrita pela etiqueta.

Alguns dos pricipais marcadores lógicos são:

<pre>
Apresenta o texto da mesma forma que foi digitado, mantendo as quebras de linhas e os espaçamentos. Por exemplo: 
<pre> este é um
                texto pre - f o r m  a   t   a   do
            um,	dois,                          três.</pre>
<blockquote>
É usada para citações de trechos longos, com diagramação esperada como em livros, parágrafos mais compactos normalmente centralizados, Por exemplo: vejamos este típico texto Machadiano em Obra Completa, Um Capitão de Voluntários:
<blockquote> Quando eu entrava, já ele estava na cadeira de balanço. Os móveis da sala eram poucos, os ornatos raros, tudo simples. X... estendia-me a mão larga e forte; eu ia sentar-me ao pé da janela, olho na sala, olho na rua. Maria, ou já estava ou vinha de dentro. Éramos nada um para o outro; ligava-nos unicamente a afeição de X... Conversávamos; eu saía para casa ou ia passear, eles ficavam e iam dormir. Algumas vezes jogávamos cartas, às noites, e, para o fim do tempo, era ali que eu passava a maior parte destas <blockquote>
<address>
Bem auto-explicativo estas etiquetas podem ser usadas dentro de uma página, como: para maiores informações ler <address>http://www.icmc.sc.usp.br/manuals/HTML/textos.html<address> que apareceria como sendo
http://www.icmc.sc.usp.br/manuals/HTML/textos.htm
<cite>
Para citações bibliográficas, de obras, textos, como a que fiz sobre Machado de Assis acima, ou como: Ver Obras poéticas de Gonçalves Dias, por Manuel Bandeira
<em>
A ênfase de textos pode ser obtida com esta etiqueta. Produz textos tipo: enfatizados.
<strong>
Os textos em negrito são normalmente produzidos por esta etiqueta.

Formatadores físicos

Estes sim especificam e marcam a aparênccia da seqüência de caracteres no navegador. Vejamos alguns:

<i>
Itálico, ou melhor, caracteres levemente inclinados, na prática. Normalmente não são fontes especiais tipo itálico e se confundem com as enfatizadas.
<tt>
Conhecido como tipo fixo (de espaçamento fixo), a abreviação significa teletype.
<u>
É o Sublinhado. Cuidado para não confundir com os links.
<strike> ou <s>
Frase riscada.
<big>
Fonte um pouco maior.
<small>
Fonte um pouco menor.
<sub>
Construção em estilo índice, como em H2O.
<sup>
Construção em estilo expoente, como em Km2.
