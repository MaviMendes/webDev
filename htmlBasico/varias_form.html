Tipos para formulários

Os formulários são muito usados para páginas HTML ativas, ou seja, páginas que respondem, ou se configuram, de acordo com as entradas dos usuários.

A seguir mostramos alguns tipos de entradas formatadas que podemos usar com partes de um formulário. Observamos que estes vários tipos de entradas não estão respondendo pois não acrescentamos chamadas para estes. No momento estamos somente analisando as possibilidades de uso e como eles são escritos.
